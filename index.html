模块 01

模板库（Templates Library）

按分类、标签、搜索呈现经典影视/剧情模板，详情页展示结构、变量与示例脚本。

分类/推荐

电影、剧集、类型、创作者精选

支持关键词搜索与热门推荐逻辑，占位的筛选条可直接绑定后端数据。

模板详情

简介 · 场景结构 · 示例脚本

收藏/使用

模块 02

模板编辑器（Template Editor）

表单变量替换 + 高级 Prompt + 分镜脚本三段式结构，便于逐层细化。

变量替换区

表单驱动的参数填写

  • 主角姓名/性别/年龄
  • 地点、背景设定、冲突、反派
  • 结局方向：喜剧/悲剧/开放式

输入后实时同步占位符，保留 onVariableChange 回调接口。

高级 Prompt

Prompt 预览与一键优化

  • 查看/编辑完整 Prompt
  • 追加“更电影感”等修饰词
  • 增删场景结构，LLM 优化入口预留

分镜脚本

Scene Breakdown

  • 镜头描述、角色动作、对白
  • 可视化分镜列表
  • 导出为生成管线输入

技术架构

NextUI + 插件化 AI 引擎

前端采用 Next.js/React + NextUI 设计体系；后端与 AI 引擎以插件式抽象，便于快速替换模型。

前端

  • Next.js + NextUI 组件
  • Tailwind/SSR 可选
  • 状态管理：Zustand/Redux

后端

  • Node (Nest) / Python (FastAPI)
  • 用户、模板、任务、计费模块
  • S3 存储，队列管理

AI 引擎（插件）

  • LLM 引擎：Prompt → 脚本
  • Scene Engine：脚本 → 分镜
  • Video / TTS / BGM 引擎

常见问题

关于接口与交付

提前回答后续接入与扩展问题，便于团队对齐。

如何切换模型供应商？

AI 引擎采用插件化定义，只需在对应服务实现 generateVideo()、generateVoice() 等接口并在配置中切换。

如何接入登录/支付？

保留 /api/auth 与 /api/billing 路由占位，可挂载微信/Google OAuth 与 Stripe/微信支付 SDK。
